<h1>Educational Hexlet project</h1>
Simple web app with postgres, caddy

Requirements

    Docker >=  20.10

    docker-compose >= 1.17

    Make

<h3>Commands</h3>
    
    ci - запуск тестов через github actions
    local_test - запуск тестов локально
    server_up - запуск сервера с приложением(необходимо заполнить .env)
